# contactBook
仿qq通讯录，包含好友列表，登录注册，实时通讯功能

# 主要主要功能
* 登录  注册 
* 好友列表 
* 个人信息管理 
* 与好友实时对话

# 开发语言 
  * 前端<br>
    VueJs + vue-router + vue-resource + vue-validator + jquery <br>
    使用 webpack进行vuejs打包<br>
    后续 使用  socket.io  进行进行实时通话
 
  * 后端<br>
    利用nodejs搭建服务器, 
    express + mongoDB + mongoose
